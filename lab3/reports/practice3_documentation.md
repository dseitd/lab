# Документация и итоговый отчёт (Практика 3)

## Инструкция пользователя
- Вход: откройте http://localhost:3000/login, введите email и пароль, нажмите «Войти».
- Регистрация: откройте http://localhost:3000/register, введите email и пароль (≥6 символов), нажмите «Зарегистрироваться».
- Основные действия: после входа на странице «Личный кабинет» заполните поле данных и нажмите «Отправить» — система отобразит введённый текст.
- Выход: нажмите кнопку «Выйти» в кабинете; будет выполнен редирект на страницу входа.

## Краткое техническое описание
- Модули:
  - server.js — сервер Express, маршруты регистрации/входа/выхода/кабинета, работа сессий.
  - sqlite3 — подключение к БД, таблица users (email, password_hash, created_at).
  - express-session — хранение сессий (cookie httpOnly).
  - bcryptjs — хэширование паролей.
- Структура:
  - ЛР1: минимальное веб-приложение (сервер, зависимости, страницы).
  - ЛР2: документация проекта (PM06-ФИО), анализ системы и скриншоты.
  - ЛР3: данный отчёт и руководство пользователя.
- Требования к запуску:
  - Node.js и npm.
  - Установка зависимостей: npm install (при необходимости).
  - Запуск: npm start и переход по адресу http://localhost:3000/.
