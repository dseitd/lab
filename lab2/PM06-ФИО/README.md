# PM06 — Supporting Information Systems

Цель: создание отдельного репозитория, фиксация начального состояния и базовый анализ информационной системы (ЛР1: авторизация/регистрация).

Структура:
- reports/ — отчёты и аналитика
- logs/ — журналы, служебные логи (пока пусто)
- screenshots/ — скриншоты интерфейса и тестов
- system_description.md — описание системы

План публикации:
- Создать приватный репозиторий pm06-supporting-information-systems
- Импортировать текущую структуру и файлы
- Добавить скриншоты и обновлять отчёты по мере выполнения работ
