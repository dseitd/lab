# Описание информационной системы

## Назначение системы
Веб‑приложение для регистрации и авторизации пользователей с защищённым кабинетом и формой ввода данных после входа.

## Основные функции
- Регистрация нового пользователя (email + пароль, хэширование пароля)
- Вход/выход, сессии и защита маршрутов (личный кабинет)
- Ввод произвольных данных в кабинете и отображение результата

## Предполагаемые пользователи
- Конечные пользователи (студенты/преподаватели) — доступ к кабинету и вводу данных
- Администратор — поддержка и расширение функционала

## Используемые технологии
- Node.js, Express
- express-session для сессий
- bcryptjs для хэширования паролей
- sqlite3 для хранения данных
- HTML/CSS (минималистичный серый интерфейс)
