# Лабораторная работа 4 — Имитация инцидента (Service Desk)

## Содержимое
- Отчёт по инциденту: reports/service_desk_incident.md

## Цель
Сформировать навыки обработки обращений пользователей и расследования причин массовых и индивидуальных сбоев.

## Кратко
- Сценарий: утром понедельника 12 преподавателей не могут войти в систему; часть видит «Неверный пароль», часть — «Сервер недоступен».
- В отчёте: приоритизация (P1), гипотезы причин, план диагностики, карточки для индивидуального и массового инцидентов, workaround, окончательное решение, тайм‑лайн реакции и восстановления.
